Welcome

This SPA runs entirely in the browser and can authenticate to GitHub, validate repositories that contain both ghwe_config.json and ghwe_log.json, then edit and commit JSON files listed in configuration.

  • OAuth sign-in uses GitHub Device Flow so no backend is required.
  • Repository validation is based on root files ghwe_config.json and ghwe_log.json.
  • Editor offers two modes: drag/drop raw JSON and CRM-style form entry.

    About

    GitHub WebEditor is a browser-based Single Page Application for maintaining static JSON files in GitHub repositories.

    Workflow:

    • Authenticate with GitHub.
    • Select a repository that contains root files ghwe_config.json and ghwe_log.json.
    • Load configured JSON files, edit in raw or CRM mode, and commit changes.
    • Automatic append to ghwe_log.json for each successful commit action.
